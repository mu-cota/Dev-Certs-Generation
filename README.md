# Dev-Certs-Generation
Generate Certs for development or non-prod purposes

Alert: 
Always start from clean slate including CA, so all Dev CA and certs are created fresh! That means Dev CA is also destroyed!

## Folder layout
Dev-Certs-Generation - working folder
Dev-Certs-Generation/certificates - area where in progress stuff are stored
Dev-Certs-Generation/cmds.txt - quick cmds, reduce typing
Dev-Certs-Generation/config/vault-0 - where completed ie. signed certs are stored (default tls server cert)
Dev-Certs-Generation/data - placeholder not in use (for future)
